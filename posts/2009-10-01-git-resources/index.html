Git Resources

At the end of my talk on Git at the Edmonton Code Camp, somebody asked if I would put the links to the various git resources up on my website.  Here they are:

  • The git website, of course.
  • Pro GIT - an awesome book about git.  You can either order the dead tree version, or read it online.
  • Linus Torvalds talking about git.  If you’re an SVN fan-boy, or have thin skin, you might want to skip this.  “…you are ugly and stupid…”
  • A fellow by the name of Bart Trojanowksi gave an awesome talk to some Ottawa ruby group.  This is long, but worth it.
  • Jason Meredith, of Los Techies fame, has a three part series on git for the Windows Developer.  Jason has some other good git stuff on his blog too.

There you go, enjoy, and be distributed.
